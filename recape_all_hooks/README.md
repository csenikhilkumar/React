# React + Vite
what is vite ?
vite is a build tool that aims to provide a faster and leaner development experience for modern web pages.
it consists of two major parts 
1. dev server :- thats provides rich feature enhancement over native es modules for example hot module replacement
2. build command :- that bundles your code with rollup , preconfigured to output highly optimized static assets for production 

what is rollup ?
it is js module bundeler compiles small pieces of code into somthing large and more complex



what is components?
In react ,component are building blocks of the user interface . they allow you to split the ui into independent , reusable pieces that can thought a custom , self contained html elements.
